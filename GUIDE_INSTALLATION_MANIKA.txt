# 🦊 EduFamily - Guide d'installation rapide

## Ce dont tu as besoin

1. **Python** (télécharge sur python.org)
2. **Un éditeur** : VS Code gratuit (code.visualstudio.com)
3. **Une clé API Anthropic** (console.anthropic.com)

---

## Installation en 5 minutes

### Étape 1 : Ouvre un terminal

- **Windows** : Tape "cmd" dans la barre de recherche
- **Mac** : Ouvre l'application "Terminal"

### Étape 2 : Va dans le dossier du projet

```
cd Téléchargements/edufamily-backend
```

### Étape 3 : Installe les dépendances

```
pip install -r requirements.txt
```

### Étape 4 : Configure ta clé API

Crée un fichier `.env` dans le dossier avec :
```
ANTHROPIC_API_KEY=ta-nouvelle-cle-ici
```

⚠️ IMPORTANT : Génère une NOUVELLE clé sur console.anthropic.com
(l'ancienne a été exposée dans le chat)

### Étape 5 : Lance l'application

```
python app.py
```

Tu devrais voir :
```
🦊 EduFamily API démarrée!
   http://localhost:5000
```

---

## Tester que ça marche

Ouvre ton navigateur et va sur :
```
http://localhost:5000/api/health
```

Tu devrais voir :
```json
{"name": "EduFamily API", "status": "ok", "version": "1.0.0"}
```
